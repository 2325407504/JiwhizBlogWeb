This is a simple blog website I build with all open source projects, to experience the Java architecture design for Web and mobile applications. The whole system is built upon Spring Framework, using SpringMVC, Spring Security, Spring Social, Spring Data, and some other open source projects, like Mockito for testing, etc. All data is stored in a NoSQL database - MongoDB. Front end UI is using Twitter Bootstrap and AngularJS. The Web application is deployed to Cloud Foundry an open source PaaS cloud platform.
